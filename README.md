Supplementary code for the manuscript “Input-dependent directionality of interactions between cortical areas”  by F. Mastrogiuseppe, J. Carmona, B. Yu, A. Kohn, C. Machens (2026)

Instructions:

-    The code is optimized for Python 3.7, and primarily relies on commonly used packages (e.g. numpy, scipy, matplotlib)

-    Scripts corresponding to different figures are organized into separate folders. 

-    Scripts should be run with setting doCompute=1 (data and plot folders are initially empty).

-    The Code/fct_xxx.py files contain the main functions used across all scripts.

-    Notation and parameters are as in the paper.

-    In order to reproduce Fig. 7, experimental data have to be loaded in the Code/MatlabData/mat_neural_data folder. Data can be downloaded from the CRCNS database, at https://doi.org/10.6080/K0B27SHN.
